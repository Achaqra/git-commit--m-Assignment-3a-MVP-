# Superior Fits MVP

## Project Description
Superior Fits MVP is a clean e-commerce prototype for Superior Fits that allows users to browse products and complete a checkout flow.

## Vibe Coding Stack
- Next.js
- Supabase
- Antigravity

## How to Run
1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open your browser and visit http://localhost:3000.
